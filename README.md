# vue-tsx-starter

## Usage

```
yarn dev
```

or

```
yarn build
```

[Set shell environment variable](https://www.digitalocean.com/community/tutorials/how-to-read-and-set-environmental-and-shell-variables-on-a-linux-vps) `NODE_ENV=production` to build deployable assets.
